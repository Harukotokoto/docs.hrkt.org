# 経済システム

Mikuの経済システムの使い方です。ユーザーごとに所持金・銀行・アイテムが管理されます（サーバー横断で共通です）。

## 基本

- **所持金**: 普段使える残高
- 新規ユーザーには初期所持金が付与されます

## お金を稼ぐ

| 方法 | 概要 |
|------|------|
| **チャットリワード** | メッセージを送るとランダムで少額付与（スパム防止あり） |
| **デイリーボーナス** | `/daily` で24時間に1回、ランダムなボーナスを受け取れます |
| **仕事** | `/work` で1時間に1回、ランダムな仕事と報酬を得られます |

## カジノ

`/casino` で以下のゲームがプレイできます。

- **スロット**: 当選確率・配当はゲーム内の説明に従います
- **サイコロ**: 予想した目で勝負
- **コインフリップ**: 表か裏かを選択

賭け金は所持金の範囲で設定します。負けると賭け金は没収されます。

## ショップ・アイテム

- **ショップ** (`/shop`): 販売中のアイテム一覧
- **購入** (`/buy`): アイテムIDと数量を指定して購入
- **売却** (`/sell`): 持っているアイテムを売却
- **インベントリ** (`/inventory`): 自分の持ち物を確認
- **ギフト** (`/gift`): 他のユーザーにアイテムを贈る

アイテムにはレアリティ（コモン・レア・エピック・レジェンダリ）があり、取引可能かどうかはアイテムごとに異なります。

## 送金

- 所持金を他のユーザーに送る機能（コマンドはBotのヘルプまたは `/help` で確認してください）

## 注意事項

- 負の残高にはならないようになっています
- スパムや短時間の連続操作には制限がかかることがあります
