# カスタムロール {#custom-role}

ユーザーが自分用のロールを作成・編集できる「カスタムロール」機能のコマンドです。管理者が `/customrole` で登録し、ユーザーが `/role` で編集します。

---

## /customrole {#customrole}

カスタマイズ可能なカスタムロールをサーバーに登録・設定します。

| 項目 | 内容 |
|------|------|
| **必要権限** | 管理者 |

### サブコマンド一覧

| サブコマンド | 説明 | オプション |
|--------------|------|------------|
| **register** | カスタムロールを登録 | `anchor`（ロール・必須）、`required_role`（ロール）、`allow_admin_bypass`（真偽値） |
| **unregister** | カスタムロールの設定を削除 | — |
| **anchor** | アンカーロールを変更 | `anchor`（ロール・必須） |
| **required_role** | 必須ロールを設定 | `required_role`（ロール・必須） |
| **allow_admin_bypass** | 管理者バイパスを設定 | オプションあり |

::: info アンカーロール
「この下にロールが作成される」基準となるロールです。カスタムロールはアンカーの下に並びます。
:::

---

## /role {#role}

登録済みのカスタムロールを編集します（作成・名前/色/アイコン変更・削除）。

### サブコマンド一覧

| サブコマンド | 説明 | オプション |
|--------------|------|------------|
| **create** | カスタムロールを作成 | `name`（文字列・必須）、`color`（HEX 6〜7文字）、`icon`（添付） |
| **name** | ロール名を変更 | `name`（必須、最大32文字） |
| **color** | ロールの色を変更 | `color`（HEX・必須） |
| **icon** | ロールアイコンを変更 | `icon`（添付・必須） |
| **delete** | カスタムロールを削除 | — |
