# VC管理コマンド {#vcmanager}

**VC Manager**（動的ボイスチャンネル等）の設定に使うコマンドです。特定のVCに参加すると、動的チャンネルが自動作成される仕組みを登録・解除できます。いわゆるTemp Voiceです。

---

## /vcmanager {#vc-manager}

VC Managerの設定をします。

| 項目 | 内容 |
|------|------|
| **必要権限** | チャンネル管理 |

### サブコマンド

| サブコマンド | 説明 | オプション |
|--------------|------|------------|
| **register** | VCを登録 | `vc`（ボイスチャンネル・必須）、`category`（動的VCを作成するカテゴリ） |
| **unregister** | VCの登録を解除 | `vc`（必須） |
