# 経済コマンド {#economy}

所持金・銀行・アイテム・カジノなど、経済システムに関するコマンドです。多くはDMでも利用できます。詳細な仕様は [経済システム](/docs/economy) を参照してください。

---

## /balance {#balance}

所持金・銀行残高・総獲得金額・総消費金額を確認します。

| 項目 | 内容 |
|------|------|
| **オプション** | なし |
| **備考** | DM可。 |

---

## /daily {#daily}

デイリーボーナスを受け取ります。24時間に1回まで。

| 項目 | 内容 |

| **オプション** | なし |
| **備考** | DM可。 |

## /work {#work}

仕事をして報酬を得ます。1時間に1回まで。ランダムで仕事種類と報酬が決まります。

| 項目 | 内容 |
|------|------|
| **オプション** | なし |
| **備考** | DM可。 |

---

## /casino {#casino}

カジノゲームをプレイします。所持金から賭け金を差し引き、勝敗に応じて配当が付きます。

### サブコマンド

| サブコマンド | 説明 | オプション |
|--------------|------|------------|
| **slot** | スロット | `bet`（整数・必須） |
| **dice** | サイコロ（予想した目で勝負） | `bet`（必須）、`prediction`（1〜6・必須） |
| **coinflip** | コインフリップ | `bet`（必須）、`choice`（表 heads / 裏 tails・必須） |

::: tip
DM可。
:::

## /economy {#economy-admin}

経済機能の設定および管理（**サーバー管理者向け**）。経済モードの切替や、ギルドモード時の所持金操作ができます。

| 項目 | 内容 |
|------|------|
| **必要権限** | サーバー管理 |

### サブコマンド

| サブコマンド | 説明 | オプション |
|--------------|------|------------|
| **switch** | 経済システムモードを切替 | `mode`（グローバル / ギルド・必須） |
| **money add** | ユーザーの所持金を増やす | `user`・`amount`（必須） |
| **money remove** | ユーザーの所持金を減らす | `user`・`amount`（必須） |
| **money reset** | このサーバー全員の所持金をリセット | — |
| **money alladd** | このサーバー全員の所持金を一括で増やす | `amount`（必須） |
