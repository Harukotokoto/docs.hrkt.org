# コマンドリファレンス（Miku 完全ガイド） {#overview}

このドキュメントを読めば、Mikuの**全機能**と**全コマンド**を把握できます。カテゴリごとにページを分け、各コマンドの説明・オプション・使い方のポイントを記載しています。

## このドキュメントの読み方 {#how-to-read}

::: tip スラッシュコマンド
Discordで `/` を入力するとMikuのコマンド一覧が表示されます。本ドキュメントのコマンド名はスラッシュ付き（例: `/help`）で記載しています。
:::

| 用語 | 説明 |
|------|------|
| **オプション** | スラッシュコマンドで表示される引数・サブコマンド。必須の場合は「必須」と記載しています。 |
| **必要権限** | サーバーで実行する際に、ユーザー側に必要なDiscord権限。 |
| **DM可** | BotとのDMでも利用できるコマンド。 |
