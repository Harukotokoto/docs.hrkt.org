# コンテキストコマンド {#context}

メッセージやユーザーを**右クリック → 「アプリ」**から実行するコマンドです。スラッシュでは表示されません。

---

## ユーザーのアバターを取得 {#get-avatar}

右クリックしたユーザーのアバターを取得します。

| 項目 | 内容 |
|------|------|
| **対象** | ユーザー（ユーザーを右クリック） |

---

## Make it a Quote {#make-it-a-quote}

右クリックしたメッセージを名言画像にします。

| 項目 | 内容 |
|------|------|
| **対象** | メッセージ（メッセージを右クリック） |

---

## メッセージを固定/解除 {#pin-message}

メッセージをBot経由で固定表示するか、固定を解除します。Discordのピン留めとは別の「固定メッセージ」機能です。新しくメッセージが送信されると、最下部に固定したメッセージが再送信されます。

| 項目 | 内容 |
|------|------|
| **必要権限** | メッセージ管理 |
| **対象** | メッセージ（メッセージを右クリック） |

::: warning
埋め込みのみのメッセージは固定できません。テキストが含まれている必要があります。
:::

::: tip
既にそのチャンネルでメッセージが固定されている場合、固定を解除します
:::
