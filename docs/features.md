# 機能一覧

Mikuで利用できる主な機能の概要です。

## 音楽・音声

### 音声合成・読み上げ

- **VOICEVOX統合**: 日本語音声合成による読み上げ
- **複数話者対応**: 様々な声質の音声を選択可能
- **リアルタイム読み上げ**: テキストチャンネルの内容を音声で再生
- **カスタマイズ**: 話者・速度・感情などの調整

### 音楽再生

- 高速で高音質な音楽再生

## 娯楽系

### レベルシステム

- メッセージ送信・リアクションで経験値（XP）獲得
- 自動的なレベル計算と通知
- サーバー内ランキング（リーダーボード）
- 特定レベルでのロール付与（レベル報酬）

詳細は [レベルシステム](/docs/leveling) を参照してください。

### 実績システム

- メッセージ数・ログイン・時間帯など多様な実績
- 活動パターンの統計追跡
- 実績解除時の通知
- サーバーごとのカスタマイズ可能

## サーバー管理

### チケットシステム

- サポートチケット（問い合わせ対応）
- テンプレート・スタッフロールの設定
- ボタンで簡単にチケット作成
- 作成・解決履歴の管理

### ロール管理

- **カスタムロール**: ユーザーが自分で作成・カスタマイズ
- **リアクションロール**: 絵文字クリックでロール付与
- **ロールキーパー**: 退出時のロール保持
- 権限管理とセキュリティ

### モデレーション

- メッセージの一括削除・フィルタリング
- スパム・不適切コンテンツの自動検出
- 警告システム（段階的な処罰）
- モデレーション活動のログ記録

## AI・自動化

### ChatAI統合

- 複数モデル対応（例: ChatGPT、Claude、Gemini、Grok）
- カテゴリ別のモデル選択
- 会話の流れを考慮した応答
- プロンプト・パラメータの調整

### 自動化ツール

- **リマインダー**: 指定時間での通知・タスク管理
- **自動公開**: アナウンスメントチャンネルの自動公開
- **ウェルカムメッセージ**: 新メンバーへの自動挨拶

## ユーティリティ

- **引用作成（Make it a Quote）**: メッセージを引用形式に変換
- **ピン留め管理**: 重要なメッセージの固定・管理
- **スナイプ**: 削除されたメッセージの表示
- **フォーム・投票**: カスタムフォーム・リアクション投票
- **サイコロ・おみくじ**: ダイス・運勢占い
- **暗号化・数学計算**: メッセージの暗号化・復号、数式計算
- **fxtwitterの置き換え** など

## 統計・分析

### サーバー統計

- アクティブユーザー・メッセージ数の追跡
- 人気チャンネル・時間帯別の活動
- メンバー数の変化（成長率）
- 定期的な統計レポート

### ユーザー統計

- メッセージ数・ログイン回数・リアクション数
- 活動時間帯・ログインストリーク
- 実績進捗・レベルアップ履歴
- サーバー内ランキング

## 経済システム

- 所持金・銀行残高の管理
- メッセージ送信時のチャットリワード
- デイリーボーナス・仕事コマンド
- カジノ（スロット・サイコロ・コインフリップ）
- ショップ・アイテム購入・売却・ギフト

詳細は [経済システム](/docs/economy) を参照してください。
